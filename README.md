Culebra Programming Language
============================

Build
-----

```bash
git clone https://github.com/yhirose/culebra.git
cd culebra
mkdir build
cd build
cmake ..
make
./culebra ../samples/fizzbuzz.cul
```

Uknown bugs
-----------

 * Cycric reference in object
 * Unintentional reference to global variable
